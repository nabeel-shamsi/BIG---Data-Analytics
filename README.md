# BIG---Data-Analytics
This repository is about how I performed BIG Data Analytics over a dummy dataset to evaluate how today's market and analysis works for BIG data.

# Big Data Analytics Pipeline using Hadoop, Spark, Docker & MySQL

## Project Overview

This project demonstrates a complete Big Data Analytics pipeline built using modern data engineering and analytics technologies.

The project covers:

- Distributed storage using Hadoop HDFS
- Containerized infrastructure using Docker
- Distributed analytics using Apache Spark
- SQL-based analysis using MySQL
- Data visualization preparation for Tableau / Power BI
- End-to-end big data workflow implementation

---

## Tech Stack

| Technology | Purpose |
|------------|---------|
| Hadoop HDFS | Distributed Storage |

| Docker | Containerization |
| MySQL | SQL Analytics |
